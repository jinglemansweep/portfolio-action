## My Setup

Inspired by [uses.tech](https://uses.tech), here's a rundown of the tools and
gear I rely on every day.

### Editor & Terminal

- **VS Code** with the Catppuccin theme and JetBrains Mono font
- **Warp** terminal with custom keybindings
- **tmux** for session management
- **Neovim** for quick edits

### Dev Tools

- **Docker Desktop** for local container workflows
- **k9s** for Kubernetes cluster management
- **HTTPie** for API exploration
- **GitHub Copilot** as a coding companion

### Hardware

- MacBook Pro 16" M3 Max
- LG 5K UltraFine monitor
- Keychron Q1 mechanical keyboard
- Logitech MX Master 3S mouse

### Productivity

- **Obsidian** for notes and knowledge management
- **Linear** for project tracking
- **Raycast** as an app launcher and clipboard manager
- **Arc** browser
